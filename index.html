Aperture

Aperture is a webapp that makes it easy to use the SentiBank API. See it in action

Build

ant war # creates dist/aperture.war
ant clean

Deploy

Copy the .WAR file to the webapps folder of a servlet container, such as Jetty, Tomcat.
